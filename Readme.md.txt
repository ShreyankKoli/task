# Task Management App

A simple task management app built with React, Node.js, and PostgreSQL.

## Features

- Add, list, and filter tasks
- Update task status (Pending → In Progress → Completed)
- Priority filtering and due date-based filtering
- Form validations on task creation

## Tech Stack

- Frontend: React
- Backend: Node.js + Express.js
- Database: PostgreSQL
- Gateway: Node API Gateway

## Setup Instructions

### Prerequisites

- Node.js and npm
- PostgreSQL
- Git

### Backend Setup

```bash
cd backend
npm install
node index.js
